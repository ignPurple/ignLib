# ignLib

A useful library to help you develop your own custom plugins easier a faster!

## Maven

```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.github.ignPurple</groupId>
    <artifactId>ignLib</artifactId>
    <version>1.0</version>
</dependency>
```
